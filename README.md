# 💊 ESP32 Bluetooth Medicine Reminder Controller

This project is a smart IoT-based medicine reminder system that integrates an Android application with an ESP32 microcontroller using Bluetooth communication. It allows users to easily manage their medication schedule and receive timely alerts.

## 🚀 Key Features

* 📱 Android app with built-in controller
* 🔵 Bluetooth connectivity with ESP32
* ➕ Add, ✏️ Edit, ❌ Delete medicines
* ⏰ Set alarms using an in-app clock interface
* 🔁 Flexible scheduling:

  * Daily reminders
  * Custom day selection
* 📋 Select and manage multiple medicines
* 🔔 Real-time alert triggering via ESP32 (buzzer/LED)

## ⚙️ How It Works

1. User connects the Android app to ESP32 via Bluetooth.
2. Medicines and schedules are configured in the app.
3. Alarm data is transmitted to the ESP32.
4. ESP32 processes the schedule and triggers alerts at the correct time.

## 🛠️ Tech Stack

* Android (Java/Kotlin)
* ESP32 (Arduino / Embedded C)
* Bluetooth Communication
